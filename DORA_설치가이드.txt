================================================================
  DORA 설치 및 사용 가이드
  Document Organizer & Retrieval Assistant
  ABC Group Tech 내부용 | 2026.04
================================================================

DORA는 Claude, ChatGPT, NotebookLM 등 AI 도구에서 다운로드한
PDF / PPTX 파일을 자동으로 분류하고 지정한 폴더로 이동해주는
Chrome 확장 프로그램입니다.


----------------------------------------------------------------
  STEP 1. Chrome 확장 프로그램 로드
----------------------------------------------------------------

1) Chrome 주소창에 아래를 입력하고 Enter
   chrome://extensions

2) 화면 우측 상단 [개발자 모드] 토글을 ON으로 켜기

3) [압축해제된 확장 프로그램을 로드합니다] 버튼 클릭

4) DORA 폴더 전체를 선택
   (예: C:\Users\이름\DORA  또는  /Users/이름/DORA)

5) 카드가 생성되면 하단에 표시된 ID를 복사해두기
   (영문 소문자 32자리 ex. abcdefghijklmnopqrstuvwxyz123456)

   ※ 이 ID는 STEP 2에서 사용합니다.


----------------------------------------------------------------
  STEP 2. 설치 프로그램 실행
----------------------------------------------------------------

  [ macOS 사용자 ]
  1) DORA_설치_mac.zip 압축 해제
  2) DORA 설치.app 더블클릭
     ※ "확인되지 않은 개발자" 경고가 뜨면:
        앱을 우클릭 → [열기] → [열기] 클릭

  [ Windows 사용자 ]
  1) DORA_설치.exe 더블클릭
     ※ "Windows의 PC 보호" 경고가 뜨면:
        [추가 정보] 클릭 → [실행] 클릭

3) 설치 화면이 열리면 STEP 1에서 복사한 Extension ID 입력

4) [설치 시작] 버튼 클릭

5) "설치 완료" 메시지 확인 후 Chrome 완전 종료 후 재시작
   (창 닫기가 아닌, Chrome을 완전히 종료해야 합니다)


----------------------------------------------------------------
  STEP 3. 저장 폴더 설정
----------------------------------------------------------------

1) Chrome 툴바에서 DORA 아이콘 클릭

2) [AI 출처 관리] 섹션에서 각 출처 카드의 [폴더 선택] 클릭
   - Claude, ChatGPT, NotebookLM 각각 설정 가능

3) 원하는 폴더 선택 (예: 바탕화면 > AI자료 > Claude)

4) 새로운 AI 도구 추가 시 [+ 추가] 버튼 클릭
   - 이름 입력 (예: Gemini)
   - URL 패턴 입력 (예: gemini.google.com)
   - 접두어 입력 (예: GMN)
   - [폴더 선택]으로 저장 위치 지정
   - [저장] 클릭


----------------------------------------------------------------
  STEP 4. 사용 방법
----------------------------------------------------------------

별도 작업 없이 평소처럼 AI 도구에서 파일을 다운로드하면
자동으로 처리됩니다.

  예시)
  Claude에서 회사소개서.pdf 다운로드
  →  파일명이 CDL_20260410_회사소개서.pdf 로 변경됨
  →  지정한 Claude 폴더로 자동 이동

  파일명 규칙:
  CDL_20260410_원본파일명.pdf
  │    │        └── 원본 파일명
  │    └────────── 다운로드 날짜 (YYYYMMDD)
  └─────────────── 출처 코드 (CDL=Claude, GPT=ChatGPT, NLM=NotebookLM)

  처리 대상 파일: .pdf, .pptx


----------------------------------------------------------------
  팝업 기능 안내
----------------------------------------------------------------

  DORA 아이콘 클릭 시 팝업이 열립니다.

  [ON/OFF 토글]    전체 기능 활성화 / 비활성화
  [루트 폴더명]    저장 경로 미지정 시 Downloads 내 생성될 폴더명
  [ETC 처리 토글]  AI 출처로 인식되지 않은 파일도 처리할지 여부
  [최근 다운로드]  처리된 파일 최근 20건 로그 확인
  [로그 초기화]    로그 전체 삭제


----------------------------------------------------------------
  문제 해결
----------------------------------------------------------------

  Q. 파일이 자동으로 이동되지 않아요.
  A. STEP 2 설치가 완료된 후 Chrome을 완전히 재시작했는지
     확인해주세요. 툴바에서 DORA 아이콘이 보여야 합니다.

  Q. "폴더 선택" 버튼이 동작하지 않아요.
  A. 설치 프로그램(STEP 2)이 정상적으로 완료되지 않은 경우입니다.
     설치 프로그램을 다시 실행해주세요.

  Q. 확장 아이콘이 보이지 않아요.
  A. Chrome 툴바 우측의 퍼즐 아이콘(🧩) 클릭 후
     DORA 옆 핀 아이콘을 클릭하면 고정됩니다.

  Q. 파일명은 바뀌는데 폴더 이동이 안 돼요.
  A. [폴더 선택]으로 저장 경로가 설정되어 있는지 확인하세요.
     경로가 비어있으면 Downloads/AI출력물/ 하위에 저장됩니다.
