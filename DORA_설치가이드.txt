================================================================
  DORA 설치 및 사용 가이드
  Document Organizer & Retrieval Assistant
  ABC Group Tech 내부용 | 2026.04
================================================================

DORA는 Claude, ChatGPT, NotebookLM 등 AI 도구에서 다운로드한
PDF / PPTX 파일을 자동으로 분류하고 지정한 폴더로 이동해주는
Chrome 확장 프로그램입니다.

zip 파일 안에는 아래 항목이 모두 포함되어 있습니다.
  - 설치 앱 (DORA_installer.app / DORA_installer.exe)
  - Chrome 확장 파일 (DORA_extension 폴더)
  - 이 가이드 파일


----------------------------------------------------------------
  STEP 1. Chrome 확장 프로그램 로드
----------------------------------------------------------------

1) 압축 해제된 폴더 안의 DORA_installer 클릭
   1.1) DORA_installer에서 설치 시작 클릭
      ※ "악성 코드가 없음을 확인할 수 없습니다" 또는
         "확인되지 않은 개발자" 경고가 뜨면:

      [방법 1] 앱을 우클릭 → [열기] → [열기] 클릭

      [방법 2] 위 방법으로도 안 열릴 경우:
         → 시스템 설정 → 개인 정보 보호 및 보안
         → 하단의 "DORA_installer가 차단되었습니다" 옆
            [그래도 열기] 클릭 → Mac 비밀번호 입력

  [ 윈도우 사용자 ]
   1.1) DORA_installer.exe 더블클릭
      ※ "Windows의 PC 보호" 경고가 뜨면:
         [추가 정보] 클릭 → [실행] 클릭

   1.2) 설치 화면이 열리면 [설치 시작] 버튼 클릭

2) chrome://extensions 클릭

3) 화면 우측 상단 [개발자 모드] 토글을 ON으로 켜기

4) [압축해제된 확장 프로그램 로드] 버튼 클릭

5) DORA_extension 폴더 선택

6) Chrome 툴바에 DORA 아이콘 고정
   → 툴바 우측 퍼즐 아이콘 클릭 → DORA 옆 핀 아이콘 클릭


----------------------------------------------------------------
  STEP 2. 설치 및 프로그램 실행
----------------------------------------------------------------
   1) "설치 완료" 메시지 확인 후 Chrome 재시작 여부를 묻는 창이 뜸
      → [예] 클릭 시 Chrome이 자동으로 완전 종료 후 재시작됨
      → [아니오] 클릭 시 직접 Chrome을 완전 종료 후 재시작 필요
         맥    → 메뉴바 Chrome → 종료  또는  Cmd+Q
         윈도우 → 우클릭 작업표시줄 Chrome 아이콘 → 창 모두 닫기


----------------------------------------------------------------
  STEP 3. 저장 폴더 설정
----------------------------------------------------------------

1) Chrome 툴바에서 DORA 아이콘 클릭

2) [AI 출처 관리] 섹션에서 각 출처 카드의 [폴더 선택] 클릭
   - Claude, ChatGPT, NotebookLM 각각 설정 가능
   ※ 클릭 후 폴더 선택 창이 뜨는 데 1~2초 걸릴 수 있습니다.
   ※ 창이 화면 뒤에 숨어있으면 Dock/작업표시줄에서 클릭하세요.

3) 원하는 폴더 선택 (예: 바탕화면 > AI자료 > Claude)

4) 새로운 AI 도구 추가 시 [+ 추가] 버튼 클릭
   - 이름 입력 (예: Gemini)
   - URL 패턴 입력 (예: gemini.google.com)
   - 접두어 입력 (예: GMN)
   - [폴더 선택]으로 저장 위치 지정
   - [저장] 클릭


----------------------------------------------------------------
  STEP 4. 사용 방법
----------------------------------------------------------------

별도 작업 없이 평소처럼 AI 도구에서 파일을 다운로드하면
자동으로 처리됩니다.

  예시)
  Claude에서 회사소개서.pdf 다운로드
  →  파일명이 CDL_20260410_회사소개서.pdf 로 변경됨
  →  지정한 Claude 폴더로 자동 이동

  파일명 규칙:
  CDL_20260410_원본파일명.pdf
  │    │        └── 원본 파일명
  │    └────────── 다운로드 날짜 (YYYYMMDD)
  └─────────────── 출처 코드 (CDL=Claude, GPT=ChatGPT, NLM=NotebookLM)

  처리 대상 파일: .pdf, .pptx


----------------------------------------------------------------
  팝업 기능 안내
----------------------------------------------------------------

  DORA 아이콘 클릭 시 팝업이 열립니다.

  [ON/OFF 토글]    전체 기능 활성화 / 비활성화
  [루트 폴더명]    저장 경로 미지정 시 Downloads 내 생성될 폴더명
  [ETC 처리 토글]  AI 출처로 인식되지 않은 파일도 처리할지 여부
  [최근 다운로드]  처리된 파일 최근 20건 로그 확인
  [로그 초기화]    로그 전체 삭제


----------------------------------------------------------------
  문제 해결
----------------------------------------------------------------

  Q. 파일이 자동으로 이동되지 않아요.
  A. 아래 순서로 확인해주세요.
     1) STEP 3에서 출처별 저장 폴더가 지정되어 있는지 확인
        (경로 미지정 시 Downloads/AI출력물/ 에 이름만 바꿔 저장됨)
     2) STEP 2 설치 완료 후 Chrome을 완전히 종료했다가 재시작했는지 확인
     3) 위 두 가지를 확인했는데도 안 되면 DORA_installer를 다시 실행하세요.

  Q. "폴더 선택" 버튼을 눌러도 창이 안 열려요.
  A. STEP 2 설치가 정상 완료되지 않은 경우입니다.
     DORA_installer를 다시 실행하고 [설치 시작]을 클릭하세요.

  Q. "폴더 선택 창이 이미 열려 있습니다" 메시지가 떠요.
  A. 이전에 연 폴더 선택 창이 아직 열려 있는 상태입니다.
     화면 뒤에 숨어있을 수 있으니 아래를 확인하세요.
     맥    → Dock에서 Python 아이콘 클릭
     윈도우 → 작업표시줄에서 해당 창 클릭
     선택을 완료하거나 취소한 뒤 다시 시도하세요.

  Q. 폴더 선택 창이 화면에 안 보여요.
  A. 창이 뒤에 숨어있을 수 있습니다.
     맥    → Dock에서 Python 아이콘 클릭
     윈도우 → 작업표시줄에서 해당 창 클릭

  Q. 확장 아이콘이 툴바에 보이지 않아요.
  A. Chrome 툴바 우측의 퍼즐 아이콘 클릭 후
     DORA 옆 핀 아이콘을 클릭하면 고정됩니다.

  Q. 맥에서 "악성 코드 확인 불가" 또는 "확인되지 않은 개발자" 경고가 떠요.
  A. [방법 1] 앱을 우클릭 → [열기] → [열기] 클릭 (처음 한 번만 필요)
     [방법 2] 시스템 설정 → 개인 정보 보호 및 보안 → 하단
              "DORA_installer가 차단되었습니다" → [그래도 열기] → 비밀번호 입력

  Q. 윈도우에서 설치 파일이 차단돼요.
  A. [추가 정보] → [실행] 을 클릭하세요.
     사내 배포 파일이므로 안전합니다.

  Q. 파일명은 바뀌는데 폴더 이동이 안 돼요.
  A. STEP 3에서 [폴더 선택]으로 저장 경로가 설정되어 있는지 확인하세요.
     카드에 경로가 표시되어 있어야 자동 이동이 됩니다.
